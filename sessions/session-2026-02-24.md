- WHERE vs HAVING


## What I understood
- 
MAX returns the highest value in a column.

AVG calculates the average number of years employed per role when used with GROUP BY role.

SUM adds all years employed per building when grouped by building.

COUNT(*) counts rows:

    - With WHERE role = "Artist" → counts only Artists.

    - With GROUP BY role → counts how many employees exist per role.

HAVING filters grouped results (after aggregation), unlike WHERE, which filters rows before grouping.

COUNT(DISTINCT shape) counts unique shapes.

STDDEV(DISTINCT weight) calculates standard deviation considering only distinct weight values.

GROUP BY shape groups rows with the same shape.

HAVING SUM(weight) < 4 filters grouped results based on total weight.

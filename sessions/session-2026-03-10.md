## What I understood
Set operators allow combining results from multiple SELECT queries.
- UNION merges results and removes duplicate rows automatically.
- UNION ALL merges results but keeps duplicates.
- MINUS returns rows that appear in the first query but not in the second.
- INTERSECT returns rows that appear in both queries.
- Set operations require the same number of columns and compatible data types.
- ORDER BY can only be applied at the end of the combined query.
